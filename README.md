# Logger

# Objetivo
Projeto criado somente para estudo de eventos de teclado, análise de entrada de usuário e compreensão de bibliotecas Python relacionadas a input handling. O uso é restrito ao ambiente local, com consentimento e sem qualquer finalidade ofensiva.

# Aviso Legal
Este projeto não deve ser usado para capturar dados de terceiros, senhas, mensagens ou qualquer informação sensível.
Capturar teclas sem autorização é crime.
Use apenas para pesquisa, aprendizado e testes locais.

# Descrição
O programa monitora eventos de teclado na própria máquina e registra para estudo de:
- funcionamento de event listeners
- interação com bibliotecas de input
- conceitos de logging e auditoria local
Não envia dados para rede, nuvem ou terceiros.

# Requisitos
Python 3.x
Bibliotecas usadas para capturar eventos (ex.: pynput, se aplicável)

# Uso
Execute o script em ambiente controlado.
Analise os eventos capturados para compreender o funcionamento.
Encerre com CTRL+C.
